üçé Mac Installation

  1. Download the .p12 file
  2. Double-click the file in Finder
  3. Enter the password when prompted:
    • Display certs: display123
    • Sales certs: sales123
  4. Choose "System" keychain
  5. Click "Add"
  6. In Keychain Access, find the certificate and set trust to "Always Trust"

üíª Windows Installation

  1. Download the .p12 file
  2. Double-click the file
  3. Choose "Local Machine"
  4. Enter the password when prompted
  5. Choose "Personal" certificate store
  6. Complete the wizard

üì± iPad Installation

  1. Email the .p12 file to yourself
  2. Open Mail on iPad and tap the attachment
  3. Tap "Install Profile"
  4. Enter the password
  5. Tap "Install" and enter iPad passcode
  6. Tap "Done"

üåê Browser Setup

After installation:

  1. Navigate to https://pixie-viewer.mardigrasworld.com
  2. Browser will prompt to select a certificate
  3. Choose your certificate from the list
  4. Click "OK" or "Continue"
Troubleshooting
  • Wrong Password: Check certificate type (display vs sales)
  • File Won't Open: Ensure you downloaded the .p12 file, not .crt
  • Browser Doesn't Prompt: Certificate may not be in the right store
  • Access Denied: Certificate may be expired or revoked
Security Notes
  • Keep certificate passwords secure
  • Don't share .p12 files with unauthorized users
  • Certificates have expiration dates
  • Contact IT if you suspect certificate compromise
